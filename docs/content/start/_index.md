doma is a CLI for tagging directories so you can recall, browse, or run commands across them. Three short pages here will take you from zero to a working install with the shell wrapper in place.

- **[Installation](./installation/)** — Homebrew, source, prebuilt binary, container.
- **[Quickstart](./quickstart/)** — `add`, `cd`, `list`, `run` in one walkthrough.
- **[Shell integration](./shell-integration/)** — wire `doma cd` so it actually changes directory.
